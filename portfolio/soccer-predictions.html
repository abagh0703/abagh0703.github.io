Soccer Match Predictions Site

This was a website to show predictions for upcoming Football (Soccer in the USA) games in the Premier League. I worked on the front end and back end, while my friends worked on the ML model and the back end. We used Flask for the back end, and a WordPress template for the front end. (We heavily modified the template's HTML and Javascript and truly pushed WordPress to do things it was not made to do.)

  • Source Code: GitHub
  • Stack: Vanilla HTML/CSS, Flask, and PostgreSQL.
